## create-react-app
하나의 명령을 실행해서 React web app을 set up 할 수  있게 해줌.

1. cmd창에 cd Documents, npx create-react-app `movie_app_2021`(폴더명)를 차례로 입력
2. vs code에서 해당 폴더를 열기
3. terminal 창에서 npm start 입력하면 React App 프로그램이 실행됨
